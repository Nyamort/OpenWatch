# Technical Specifications - Mail Detail Analytics

## 1. Data retrieval
- Filter mail messages by class and period.
- Sort descending by timestamp.

## 2. Sections
- Header charts maintain same buckets/time-window behavior as mail list.
- Message table includes `source`, `mailer`, `subject`, recipient counters and duration.
- No row-level drilldown action in MVP; detail remains single-screen scope.
