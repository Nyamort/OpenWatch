# Technical Specifications - Command Run Detail Analytics

## 1. Scope and retrieval
- Resolve run by command execution ID and environment scope.
- Include exception summary, logs, query and event counters through correlation keys.

## 2. Timeline and metadata
- Build timeline from normalized execution stages.
- Include run header, issue affordance, and status badge styling.

## 3. Performance and links
- Keep request log/exception links to dedicated pages where available.
- Back link returns to command detail with filters intact.
