# Technical Specifications - Outgoing Request Detail Analytics

## 1. Scope and retrieval
- Detail context: selected host/domain and active filters.
- Pull matched requests by host and timestamp window.

## 2. Sections
- Two charts remain host-scoped and period-scoped.
- Request list supports duration + status segmented filters.

## 3. UX
- Request details preserve execution-context badges and source info.
- Back navigation returns to host-scoped list with context.
